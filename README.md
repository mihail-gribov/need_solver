# Need Solver

Survey engine with inference based on 4-valued fuzzy logic (Belnap-Lukasiewicz).

## What it does

- Runs adaptive questionnaires to collect user needs and constraints
- Infers best-matching objects from catalog using fuzzy CNF formulas
- Handles uncertainty with 4 truth values: TRUE, FALSE, UNKNOWN, CONFLICT
- Provides explainable rankings with per-criterion breakdown
